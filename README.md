`make`

`./so_long maps/map2.ber`

* Have a working history.

* Implement redirections:
  - < and > redirect input/output
  - << and >>
* Implement pipes.

* Handle:
  - environment variables
  - $? (exit status)
  - ctrl-C, ctrl-D
  - echo, cd, pwd, export, unset, env, exit
